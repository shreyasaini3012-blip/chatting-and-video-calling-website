# chatting-and-video-calling-website

A simple real-time chatting web app built using Node.js and Socket.io, allowing multiple users to send and receive messages instantly.

🚀 Features

- Real-time messaging
- Multiple users support
- Lightweight and fast

🛠 Tech Stack

- HTML, CSS, JavaScript
- Node.js, Express
- Socket.io

▶️ Run Locally

npm install
node server.js

Open: http://localhost:3000/chat.html
